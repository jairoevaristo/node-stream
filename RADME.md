# Streams in NodeJS

## Description
A video is at the root of the project a `video` folder is served to an HTML page, via an` express` server 

## Get started

1 - Install the necessary dependencies

```bash
  npm install
```
or 
 
```bash
  yarn install
```

2 - Run the command to start the server

```bash
  npm run dev
```

or 

```bash
  yarn install
```
